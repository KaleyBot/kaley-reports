# Summary: What Education/Training Do These Jobs Require?

## Degree Requirements (from most common to least)
1. **Bachelor's in Computer Science** - Most common requirement for engineering roles
2. **Master's in CS/ML** - Preferred for senior/research roles, increasingly common
3. **PhD** - Required/hard requirement for research engineer roles
4. **Equivalent experience** - Many roles accept strong portfolio/proven skills over degree
5. **No degree required** - Entry-level roles like AI Training Specialist, Interns

## Certifications That Help (from most valuable)
1. **Cloud Certifications** (AWS, GCP, Azure) - Highly valued for MLOps, engineering roles
2. **ML/AI Certifications** (DeepLearning.AI, Coursera, Google ML) - Show commitment
3. **Project Management** (PMP) - Helpful for delivery/manager roles
4. **Cloud Security** - For roles handling sensitive AI data

## Key Skills Most Employers Want
- **Python** (essential for almost all AI roles)
- **LLM APIs & Prompt Engineering** (hottest skill right now)
- **LangChain/LangGraph** (framework experience)
- **Vector Databases** (Pinecone, Weaviate, etc.)
- **Cloud Platforms** (AWS, GCP, Azure)
- **MLOps/Deployment** (getting models to production)
- **Agentic Workflows** (autonomous AI systems)

## Recommendation for Education Path
Based on these job listings:

1. **Start with:** Python programming skills (free to learn)
2. **Build foundation:** CS degree or equivalent (many online options)
3. **Specialize in:** AI/ML (through certificates, bootcamps, or master's)
4. **Get practical experience:** Build projects, contribute to open source
5. **Consider certifications:** AWS cloud certs are widely recognized
